# Bus Management System

## Description
This is a Bus Management System developed using PHP and MySQL.
It allows users to register, login, book tickets, select seats, and view booking history.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS

## Features
- User Registration & Login
- Bus Listing
- Seat Selection
- Ticket Booking
- Booking History
- Dashboard

## How to Run the Project

1. Install XAMPP.
2. Place the project folder inside htdocs.
3. Start Apache and MySQL.
4. Open phpMyAdmin.
5. Create a database named: busmanagement
6. Import the provided SQL file.
7. Open browser and go to:
   http://localhost/busmanagement/
